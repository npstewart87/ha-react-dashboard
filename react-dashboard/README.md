# React Dashboard for Home Assistant

A fully customizable dashboard for [Home Assistant](https://www.home-assistant.io/) built with React.

## Features

- Modern React-based UI
- Fully customizable panels (lights, climate, sensors, cameras, and more)
- Ingress support for seamless HA integration
- Real-time entity state updates via WebSocket

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the **⋮** menu (top right) → **Repositories**
3. Add: `https://github.com/npstewart87/ha-react-dashboard`
4. Find **React Dashboard** in the store and click **Install**

## Configuration

No configuration is required. The add-on uses ingress to integrate directly into the Home Assistant sidebar.
